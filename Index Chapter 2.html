Operators In JavaScript 

        
    Operators In JavaScript
    Used to perform some operation on "Data"

    Artihmetic Operators
            Artihmetic Operators are "+, -, *, /"
            * Modules --> %
            * Exponentiation --> a2 or, 22 We use 2 "**" for Exponentiation operator for appliation.
                    Exmple: "5^2 = 5x5 = 25"
            * Increment (Unary Oprater) --> ++
                    Example : if i type a++ or a+1 both have the same porperties
                        a++ is valid and ++a is also valid.
                        a++ is called post Increment (post mean after) in post Increment the value will not change in this line it will change from the next line.
                        ++a is called pre Increment (pre mean before) in pre Increment the value will change first then work will progress furhter.
            * Decrement (Unary Oprater) --> --
                    Example : if i type a-- or a-1 both have the same porperties
                        a-- is valid and --a is also valid.
                        a-- is called post Decrement (post mean after) in post Increment the value will not change in this line it will change from the next line.
                        --a is called pre Decrement (pre mean before) in pre Increment the value will change first then work will progress furhter.
