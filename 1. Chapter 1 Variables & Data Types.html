Defination of let,const and var
            var: Variable can be re-declared & updated. A global scope (can be used anywhere in the whole code) variable.
            let: Variable cannot be re-declared but can be updated. A block scope ({code}(one block) are block) variable.
                    block example:
                    {let a =5
                    console.log (a)
                    }
                    {let a = 10
                    console.log(a)
                    }
                    the both values will be print.
                Note: if we define a variable without any value it will print undefined.
                        Example : let a;
                              console.log(a)
                              the answer will be undedifined
            const: variable cannot be re-declared or updated. A block scope variable.
                Note: we must give const a value other wise it will give the error.
                      wihtout any value it will not be printed or even show null or undedifined.

we will use let & const in our most casses 

example: let name = "Qamar Ali"
            The result will printed as the string text 
            in this case the string text is Qamar Ali.
